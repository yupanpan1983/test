OpenClaw 是什么？

一个 开源的自托管网关，将聊天应用连接到 AI 代理。
在你的机器上运行 Gateway，它就成为消息应用与 AI 助手之间的桥梁。

🎯 为谁而生

开发者和高级用户——想要可以随时随地通过消息交互的个人 AI 助手，不依赖托管服务。

⚡ 核心差异

自托管（你的规则）· 多渠道（一个服务所有平台）· Agent 原生（不只是聊天机器人）。

🔒 隐私优先

数据完全在本地，无需第三方云服务。你拥有完全的控制权和访问权限。

🌐 开源生态

GitHub 开源，社区驱动，插件化架构，支持自定义扩展和二次开发。

部署方式

🏠 本地部署

在自己电脑上运行，数据完全本地，延迟最低。

npm i -g openclaw
openclaw onboard
openclaw gateway start

☁️ VPS / 云服务器

7×24 在线，支持 Tailscale 内网穿透，随时响应。

openclaw gateway install
openclaw gateway start

Step 2 · 引导配置

Step 3 · 启动

openclaw gateway start
